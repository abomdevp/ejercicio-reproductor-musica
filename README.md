# ejercicio-reproductor-musica

El ejercicio está aplicado con la metodología BEM, OOCSS y SMACSS.
BEM me permite nombrar las clases y dividir el HTML en bloques, elementos y modificadores.
OOCSS me permite separar estructura para reutilizar código.
SMACSS me permite ordenar el CSS en categorías.

La clase theme-retro está establecida con la fuente Roboto y con efecto bold.
La clase is-paused está por defecto (y se puede cambiar por is-playing según el estado del reproductor).
Las clases btn-control y skin-primary o secondary están adaptadas a los principios de OOCSS.
